# ATS Resume Maker

**Pembuat resume ATS-friendly yang gratis, modern, dan mudah digunakan**

ATS Resume Maker adalah aplikasi web yang dirancang untuk membantu Anda membuat resume yang optimal untuk sistem Applicant Tracking (ATS). Dibangun dengan teknologi modern, aplikasi ini menyediakan pengalaman pengguna yang luar biasa dengan editor real-time, multiple template pilihan, dan fitur-fitur canggih untuk memaksimalkan peluang Anda dalam proses rekrutmen.

### Editor & Preview
- **Live Preview**: Pratinjau resume Anda secara real-time saat mengetik
- **Zoom Controls**: Atur ukuran tampilan preview dari 25% hingga 100% untuk kenyamanan membaca

## ğŸ“‹ Cara Penggunaan

### 1. Memulai Resume Baru

Ketika pertama kali membuka aplikasi, Anda akan melihat tampilan kosong dengan tab-tab berbeda untuk setiap seksi resume.

### 2. Tab Personal Information

**Isi data diri Anda:**
- **Full Name**: Nama lengkap Anda (akan tampil sebagai header)
- **Email**: Email yang dapat dihubungi
- **Phone**: Nomor telepon
- **Location**: Kota/negara tempat tinggal
- **Professional Summary**: Ringkasan karir Anda (2-3 baris tentang pengalaman & tujuan)
- **LinkedIn**: Username atau URL LinkedIn (opsional)
- **GitHub**: Username atau URL GitHub (opsional)

**Tips:**
- Professional Summary harus ringkas namun impactful
- LinkedIn & GitHub ditampilkan sebagai link di header resume

### 3. Tab Experience

**Tambah pengalaman kerja Anda:**
- **Position**: Jabatan/posisi yang Anda pegang
- **Company**: Nama perusahaan
- **Location**: Kota/negara perusahaan (opsional)
- **Start Date**: Bulan/tahun mulai (format: Jan 2020)
- **Current**: Centang jika Anda masih bekerja di perusahaan ini
- **End Date**: Tanggal selesai (muncul otomatis jika tidak "Current")
- **Bullets**: Pencapaian/tanggung jawab dalam bullet points

**Menambah Bullet Points:**
- Klik "Add Bullet" untuk menambah bullet point baru
- Tulis achievement yang spesifik dan terukur
- Gunakan action verbs seperti "Developed", "Implemented", "Led", dll
- Highlight impact: result/metrics jika ada (contoh: "Increased sales by 25%")

**Mengurutkan Pengalaman:**
- Drag dan drop pengalaman untuk mengubah urutan
- Biasanya urutan dari yang paling baru ke yang paling lama

**Tips Penulisan Bullet:**
- Gunakan format: [Action Verb] [Task] [Result/Impact]
- Contoh: "Led team of 5 developers to deliver project 2 weeks ahead of schedule"
- Hindari pronoun seperti "I", "We", "The"
- Buat setiap bullet jelas dan relevan dengan posisi yang dituju

### 4. Tab Education

**Tambah riwayat pendidikan:**
- **Degree**: Gelar yang Anda dapatkan (S1, S2, Diploma, dll)
- **School**: Nama institusi pendidikan
- **Location**: Kota/negara sekolah (opsional)
- **Start Date**: Tahun mulai kuliah
- **End Date**: Tahun lulus
- **GPA**: Grade Point Average (opsional, hanya jika > 3.5)

**Tips:**
- Urutkan dari pendidikan tertinggi ke terendah
- GPA di atas 3.5 bisa menambah nilai, jika di bawah pertimbangkan untuk dihapus
- Anda bisa menambah lebih dari satu education entry

### 5. Tab Skills

**Daftar keahlian Anda:**
- Klik "Add Skill" untuk menambah skill baru
- Ketik nama skill (contoh: React, Python, Project Management, dll)
- Skill ditampilkan dalam satu baris dengan pemisah bullet

**Tips:**
- Prioritaskan skill yang relevan dengan lowongan pekerjaan
- Mix technical skills (React, Python, SQL) dengan soft skills (Communication, Leadership)
- 10-15 skill sudah cukup
- Urutkan dari yang paling kuat ke yang less strong

### 6. Tab Projects

**Showcase proyek Anda:**
- **Project Name**: Nama proyek yang jelas
- **Description**: Deskripsi singkat tentang apa yang dibangun (2-3 baris)
- **Project URL**: Link ke live project, repository, atau portfolio
- **Technologies**: Teknologi/tools yang digunakan (pisahkan dengan koma)

**Tips:**
- Pilih 3-5 proyek terbaik yang relevan dengan posisi
- URL akan menjadi link aktif di PDF
- Technologies ditampilkan dalam format italic kecil untuk referensi

### 7. Tab Certifications

**Tambah sertifikasi profesional:**
- **Certification Name**: Nama sertifikat (contoh: AWS Solutions Architect)
- **Issuer**: Organisasi yang mengeluarkan sertifikat
- **Date**: Tanggal diperoleh
- **URL**: Link untuk verifikasi sertifikat (opsional)

**Tips:**
- Hanya masukkan sertifikasi yang relevan dan valid
- URL sertifikat akan menjadi link "View Certificate" di PDF
- Sertifikasi menunjukkan komitmen Anda pada learning berkelanjutan

### 8. Tab Languages

**Daftar bahasa yang Anda kuasai:**
- **Language Name**: Nama bahasa
- **Proficiency Level**: Pilih dari Native, Fluent, Intermediate, Basic

**Tips:**
- Jangan cantumkan bahasa yang Anda tidak fasih
- Level yang jelas membantu employer mengevaluasi kemampuan komunikasi

### 9. Tab Links

**Tambah link portfolio dan media sosial lainnya:**
- **Label**: Nama link (Portfolio, Dribbble, Medium, Twitter, dll)
- **URL**: Link lengkap

**Quick Add Presets:**
- Portfolio, Dribbble, Behance, Medium, Twitter/X, Stack Overflow

**Tips:**
- Sangat berguna jika Anda bukan developer (designer, writer, content creator)
- Pastikan link aktif dan menampilkan karya terbaik Anda
- Portfolio link sangat valued oleh employer untuk kreatif roles

## ğŸ’¾ Saving & Exporting

### Auto-Save
- Aplikasi otomatis menyimpan setiap perubahan ke browser storage
- Data tersimpan lokal di device Anda, bukan di server

### Export PDF
1. Klik tombol **"Export PDF"** di header
2. PDF akan didownload ke device dengan nama: `[Nama Anda].pdf`
3. File berukuran 50-150KB (kecil dan efisien)
4. PDF siap dikirim ke recruiter/apply di job portal

**PDF Features:**
- ATS-friendly format dengan text selectable
- Semua URL adalah link aktif yang bisa diklik
- Times New Roman font (ATS standard)
- Responsive ke setiap template yang dipilih

### Export JSON
1. Klik tombol **"Save"** di header
2. File JSON akan didownload dengan nama: `[Nama Anda].json`
3. Gunakan untuk backup atau sharing data

### Import JSON
1. Klik tombol **"Import"** di header
2. Pilih file JSON dari device Anda
3. Data akan di-load dan mengganti current resume (bisa undo dengan clear data)

## âš™ï¸ Tips Penggunaan

### Optimasi untuk ATS
- **Gunakan section titles standar**: Experience, Education, Skills, Projects, dll
- **Hindari format khusus**: Graphics, charts, dan fancy formatting tidak bisa dibaca ATS
- **Clean layout**: Spacing yang konsisten memudahkan parsing
- **Standard fonts**: Template sudah menggunakan Times New Roman yang ATS-friendly

### Penulisan Efektif
- **Action Verbs**: Developed, Implemented, Led, Managed, Created, Designed
- **Quantify Results**: "Increased efficiency by 40%" lebih baik dari "Improved"
- **Tailoring**: Adjust bullet points sesuai job description
- **Relevant Only**: Hanya masukkan yang relevan dengan posisi

### Best Practices
1. **Satu halaman untuk junior**, maksimal 2 halaman untuk senior
2. **Recentness bias**: Pengalaman terbaru paling penting
3. **Customization**: Buat multiple versions untuk berbagai industri
4. **Proofreading**: Double-check typo dan formatting sebelum submit
5. **Testing**: Download PDF dan buka di berbagai browser/device
